# Proyecto One Piece - Grand Line Collectibles

## Propósito
Este es un sitio web de e-commerce temático dedicado a la venta de artículos coleccionables de One Piece. El objetivo es ofrecer una experiencia visual atractiva para fans, utilizando tecnologías modernas de maquetación web.

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño modular con variables y arquitectura Top-Down.
* **Flexbox:** Organización de la sección de productos.
* **CSS Grid:** Distribución de la sección de reseñas.
* **Formspree:** Gestión de envíos del formulario de contacto.
